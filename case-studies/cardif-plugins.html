El Reto

Cardif (BNP Paribas) quería expandir su negocio de seguros en LATAM ofreciendo embedded insurance — seguros integrados directamente en el checkout de e-commerce y plataformas financieras.

Los desafíos eran múltiples:

  • Integración compleja — Cada partner necesitaba desarrollo custom
  • Time-to-market lento — Meses para cada nueva integración
  • Regulaciones diferentes — Cada país tiene normativa distinta
  • Soporte fragmentado — Sin documentación unificada

La Solución

Desarrollamos un ecosistema completo de plugins embebibles:

4 Productos de Seguro como Widget:

  • Seguro Prestamista — Protección de pagos de crédito
  • Garantía Extendida — Cobertura extra para electrodomésticos
  • Seguro de Vida Crédito — Protección familiar vinculada a préstamos
  • Protección Desempleo — Cobertura de cuotas en caso de despido

Sistema de Integración Universal:

  • Web Components — Funcionan en cualquier framework (React, Vue, vanilla JS)
  • API Gateway unificada — Un endpoint, múltiples productos
  • Configuración por país — Regulaciones automáticas según mercado
  • Sandbox completo — Pruebas sin afectar producción
"Antes tardábamos 3 meses en integrar un nuevo partner. Ahora lo hacen solos en una tarde con la documentación."

El Ecosistema Completo

1. Catálogo de Plugins — Landing comercial mostrando los 4 productos con simuladores interactivos

2. Portal de Partners — Dashboard B2B para gestionar partners, asignación de plugins, métricas de uso y feed de actividad

3. Developer Docs — Documentación técnica con tester en vivo, snippets para React/JS/Angular, y referencia completa de API

4. Landing Comercial — Página de ventas B2B con logos de partners reconocidos como prueba social
